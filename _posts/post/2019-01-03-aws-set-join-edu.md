AWS Educate는 기존 계정 용량에 더하여 연 100달러의 크레딧을 추가로 받을 수 있다. (매년 갱신 가능)  
AWS Educate Starter는 실습계정과 같은 개념으로 1인당 75달러의 크레딧을 제공하며 소진 시 계정이 중지된다.  
(결제 카드 등록 X)

### AWS Educate & AWS Educate Starter

#### AWS Educate 가입
